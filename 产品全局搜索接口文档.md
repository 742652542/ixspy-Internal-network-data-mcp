# 产品全局搜索接口文档

## 一、接口概述

| 项目 | 说明 |
|------|------|
| 接口地址 | `http://etsy.int.ixspy.com/api/etsy-goods-all` |
| 请求方式 | POST |
| Content-Type | application/json |
| 接口说明 | 根据多种筛选条件搜索Etsy平台商品信息，支持关键词、价格区间、销量、收藏、评分等多维度筛选 |
| 数据限制 | 最多返回10000条数据 |

---

## 二、请求参数

### 2.1 基础参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| page | string | 否 | 1 | 页码 |
| size | string | 否 | 20 | 每页条数 |
| orderBy | string | 否 | favorites_total | 排序字段，见[排序字段枚举](#71-排序字段枚举) |
| orderType | string | 否 | desc | 排序方式：`desc`-降序 / `asc`-升序 |

### 2.2 文本搜索参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| search_content | string | 关键词搜索。支持多词搜索：`\|` 表示或关系，`&` 表示且关系。支持商品URL或商品ID搜索 |
| search_content_not | string | 标题过滤词，排除包含指定关键词的商品，多词用 `\|` 分隔 |
| store_id | string | 店铺ID |
| materials | string | 材质搜索 |
| categoryIds | array | 分类ID数组，如 `[123, 456]` |
| category_path | string | 分类路径末级ID |

### 2.3 标志位筛选（枚举类型）

以下字段具有相同的枚举值：

| 值 | 类型 | 说明 |
|----|------|------|
| `""` 或 不传 | string | 不限 |
| `1` | number | Yes（是）|
| `-1` | number | No（否）|

适用字段列表：

| 参数名 | 说明 |
|--------|------|
| best_seller | 热销标志 |
| personalization | 是否支持定制 |
| is_ad | 是否为广告商品 |
| netfly_tag | 是否已采集（加入过采集队列） |
| etsy_pick | Etsy's Pick 标记 |
| rare_find | Rare Find 标记 |

**请求示例：**
```json
{
    "best_seller": 1,
    "is_ad": -1,
    "personalization": ""
}
```

### 2.4 时间筛选（枚举类型）

#### 上架时间 `day`

| 值 | 说明 |
|----|------|
| 0 | 不限 |
| 30 | 30天内 |
| 60 | 60天内 |
| 90 | 90天内 |
| 180 | 近半年 |
| 365 | 近一年 |
| 366 | 一年以上 |

#### 最近卖出时间 `latest_sale_time`

枚举值同上架时间 `day`

#### 广告投放时间

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ad_time_start | string | 广告投放开始时间戳（秒级） |
| ad_time_end | string | 广告投放结束时间戳（秒级） |

### 2.5 24小时内购买人数 `sales_24hours`

| 值 | 类型 | 说明 |
|----|------|------|
| `0` | string | 不限 |
| `1-20` | string | 1-20人 |
| `20+` | string | 20人以上 |

## 七、附录

### 7.1 排序字段枚举

| 字段 | 说明 |
|------|------|
| favorites_total | 累计收藏数（默认）|
| reviews_total | 累计评论数 |
| sales_total | 累计销量 |
| carts_total | 累计加购数 |
| est_sales_total | 估算总销量 |
| created_time | 创建时间 |

---

### 7.2 搜索关键词说明

- **单关键词：** `"necklace"` 搜索包含 necklace 的商品
- **或关系：** `"necklace|ring"` 搜索包含 necklace 或 ring 的商品
- **且关系：** `"necklace&silver"` 搜索同时包含 necklace 和 silver 的商品
- **组合：** `"necklace|ring&silver"` 搜索包含(necklace 或 ring)且包含 silver 的商品
- **URL搜索：** 直接粘贴 Etsy 商品链接
- **ID搜索：** 输入商品ID，支持多个ID用 `\|` 分隔

---

### 7.3 使用建议

1. **分页限制：** 单次查询最多返回10000条数据，超过后需调整筛选条件
2. **性能优化：** 建议先使用关键词+分类缩小范围，再添加其他筛选条件
3. **区间参数：** 起始值和结束值为 0 时表示不限制该条件
4. **标志位参数：** 不传或传空字符串表示不限，传数字需注意类型
